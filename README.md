# Arduino Uno LED Blinker (Wokwi Simulator)

A streamlined embedded system prototype featuring an Arduino Uno, a current-limiting resistor, and an LED.

## 🛠️ Hardware Setup
* **Board:** Arduino Uno R3
* **Pin 13:** Connected to LED (+) Anode
* **Ground (GND):** Connected via 220Ω Resistor to LED (-) Cathod

## 📂 Repository Contents
 sketch.ino - Core C++ firmware logic.
diagram.json - Wokwi virtual wiring layout.
